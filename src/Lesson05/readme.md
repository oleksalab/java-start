# Урок 5 - Массивы

- данные варианты объявления массивов - равны:

```
int[] a;
int a[];
```

- после объявления массива он ещё не создан и пуст

```
String[] str;   
```

- после инициализации массив будет создан и в памяти будет выделено некоторое место

```
String[] str = new String[n];
```
